    # Migration Steps

    위 아티클에서는 JDK11을 이용하여 JAVA 애플리케이션을 만드는 것이 아니라, 기존 애플리케이션을 JDK11을 이용하여 빌드하고 JAVA11에서 돌아가도록 하는 방법을 말하고 있다.

    1. JDK11에서 기존 애플리케이션을 실행한다.
    2. JAVA11을 이용하여 애플리케이션을 빌드한다.
    3. Module System을 이용하여 애플리케이션을 모듈화 한다.
